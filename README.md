ez-snake
=========

Snake-like game for Rachel

## Play

[Play snake!](http://ez121sl-ez-snake.s3-website-us-west-2.amazonaws.com) (Link is subject to change)

## Build from source

```bash
$ lein do clean, jar
```
Then open `resources/public/index.html` in a modern browser.
